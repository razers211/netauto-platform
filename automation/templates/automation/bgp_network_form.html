BGP Network Advertisement Information
Network Advertisement

BGP network advertisement announces routes to BGP neighbors. Only networks that exist in the routing table will be advertised to neighbors.

Network Examples

Common network formats:
• 192.168.1.0 / 255.255.255.0 (Class C)
• 10.0.0.0 / 255.0.0.0 (Class A)
• 172.16.0.0 / 255.240.0.0 (Class B)

Important: Ensure the network you're advertising exists in the routing table (either as a connected route or static route) before BGP will advertise it.
