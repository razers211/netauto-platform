BGP Neighbor Configuration Information
BGP Neighbors

BGP neighbors are routers that exchange routing information. The neighbor IP should be reachable and configured to accept BGP connections from this router.

AS Numbers

AS (Autonomous System) numbers identify routing domains:
• Private AS: 64512-65534
• Public AS: Assigned by regional registries
• 4-byte AS: 1-4294967295

VRF-Aware BGP: If a VRF name is specified, the neighbor will be configured within that VRF's address family, enabling MPLS VPN functionality.
