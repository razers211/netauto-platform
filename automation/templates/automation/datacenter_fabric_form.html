Datacenter Fabric Information
EVPN VXLAN Datacenter Fabric: Automated deployment of spine-leaf architecture with EVPN control plane and VXLAN data plane. This creates a scalable, multi-tenant datacenter network with seamless Layer 2/3 connectivity.
Spine Switch

Acts as the underlay core, providing IP connectivity between leaf switches. Runs OSPF for underlay and BGP EVPN for overlay.

Leaf Switch

Connects to servers and provides VXLAN tunneling. Acts as VTEP (VXLAN Tunnel Endpoint) and BGP EVPN PE.

Border Leaf

Provides external connectivity to WAN/DCI networks. Performs EVPN-to-IP routing for tenant traffic.
