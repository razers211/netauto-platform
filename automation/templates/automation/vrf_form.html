VRF Configuration Information
What is a VRF?

Virtual Routing and Forwarding (VRF) creates multiple routing table instances on a single router, enabling network segmentation and multi-tenancy.

Route Targets

Route targets control route distribution in MPLS VPNs. Import RT determines which routes to import into the VRF, while Export RT determines which routes to export from the VRF.
