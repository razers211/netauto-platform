BGP VRF Configuration Information
BGP VRF Integration

This configuration enables BGP for a specific VRF, allowing the router to participate in MPLS VPN services and maintain separate routing tables for different customers.

Route Targets

Route targets control VPN route distribution:
• Import RT: Routes to receive into this VRF
• Export RT: Routes to advertise from this VRF
• Format: ASN:value (e.g., 65000:100)

Prerequisites: The VRF must already exist on the device before configuring BGP for it. Use the VRF creation feature first if the VRF doesn't exist.
