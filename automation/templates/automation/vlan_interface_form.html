VLAN Interface (SVI) Information
What is a VLAN Interface?

A VLAN interface (also called SVI - Switched Virtual Interface) provides Layer 3 functionality for a VLAN. It acts as a gateway for devices in that VLAN to communicate with other networks.

Use Cases
  • Inter-VLAN Routing: Enable routing between VLANs
  • Default Gateway: Act as gateway for VLAN hosts
  • Management Interface: Remote management access
  • DHCP Relay: Forward DHCP requests
Configuration Examples
Cisco Commands:
interface vlan 100
description Gateway for VLAN 100
ip address 192.168.100.1 255.255.255.0
no shutdown
Huawei Commands:
interface Vlanif100
description Gateway for VLAN 100
ip address 192.168.100.1 24
undo shutdown
quit
Prerequisites:
  • The VLAN must exist before creating the VLAN interface
  • At least one port must be assigned to the VLAN for the interface to be active
  • If using VRF, the VRF must be created first
  • The IP address should be within the network range planned for this VLAN
